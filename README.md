# Chinese Food Platform

## Project Overview

Chinese Food Platform is a web application designed to showcase traditional Chinese cuisine and regional food culture. The platform organises different cuisine categories and provides detailed information about representative dishes.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Features

### Regional Cuisine Classification

* Sichuan Cuisine
* Cantonese Cuisine
* Zhejiang Cuisine

### Dish Information Pages

* Traditional dish introductions
* Cultural background information
* Structured content presentation

### User Experience

* Easy navigation
* Clear information hierarchy
* Responsive webpage design

---

## Skills Demonstrated

* Frontend Development
* Content Management
* Information Organisation
* Responsive Web Design
* UI Design

---

## Project Outcome

Developed a user-friendly food information platform that improves content organisation and demonstrates practical web development skills.
